# 🏆 Tabela da Copa do Mundo FIFA 2026™

Bem-vindo à Tabela Dinâmica da Copa do Mundo de 2026! Este é um aplicativo web (PWA) completo que permite anotar todos os resultados do maior torneio de futebol do mundo, contando com o chaveamento automático de 32 times baseado nas regras oficiais da FIFA.

![Status](https://img.shields.io/badge/Status-Concluído-success)
![Versão](https://img.shields.io/badge/Versão-1.0-blue)

## ✨ Funcionalidades

- **Calendário Real:** Todos os jogos da fase de grupos e mata-mata estão sincronizados com as datas e horários oficiais da FIFA (convertidos para o Horário de Brasília).
- **Avanço Automático:** O sistema calcula automaticamente os 1ºs e 2ºs colocados de cada grupo, além dos 8 melhores 3ºs colocados, preenchendo as chaves do mata-mata instantaneamente.
- **Instalável (PWA):** O projeto funciona offline e pode ser instalado diretamente no celular ou computador como um aplicativo nativo.
- **Layout Responsivo:** Design moderno ("glassmorphism") dividido em abas, perfeitamente adaptado para leitura em smartphones, tablets e desktops.

## 🚀 Como usar e instalar

1. Acesse o link do projeto hospedado: **[INSERIR O LINK DO SEU GITHUB PAGES AQUI]**
2. **No Computador:** Preencha os placares e veja o sistema calcular quem avança até a Grande Final.
3. **No Celular:** Ao acessar o site, o navegador sugerirá "Adicionar à Tela Inicial" ou "Instalar Aplicativo". Aceite para ter o ícone do simulador junto com seus outros apps!

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando apenas tecnologias web puras, garantindo leveza e rapidez, sem a necessidade de frameworks pesados:

- **HTML5** (Semântica e estrutura do App e PWA)
- **CSS3** (Estilização avançada, CSS Grid, Flexbox e animações)
- **Vanilla JavaScript** (Lógica de classificação, motor de chaveamento, Service Workers e controle de LocalStorage)

## 📁 Estrutura de Arquivos

- `index.html`: A interface principal e toda a lógica de renderização embutida.
- `manifest.json`: Arquivo de manifesto que transforma o site em um aplicativo instalável.
- `sw.js`: Service Worker que intercepta as requisições, permitindo que o app funcione sem internet.
- `icon-192.png / icon-512.png`: Ícones do aplicativo para a tela inicial dos smartphones.
